# Inventory Management App

A React-based inventory management application for managing products, categories, and stock.  
This project provides a clean dashboard interface with CRUD functionality (Create, Read, Update, Delete) and modern UI components.

## 🚀 Features
- Add, edit, and delete products
- Categorize products with dynamic category selection
- Product cards with image, price, and description
- Modal form for adding and editing products
- Dashboard with key metrics and product overview
- Responsive design using Tailwind CSS

## 🛠️ Tech Stack
- **React** (functional components + hooks)
- **Tailwind CSS** (modern styling)
- **JavaScript (ES6+)**
- Future-ready for integration with APIs or localStorage persistence

## 📂 Project Structure
- `Products.js` → Product listing, modal form, CRUD logic
- `Dashboard.js` → Overview page with metrics and charts (planned)
- `App.js` → Root component
